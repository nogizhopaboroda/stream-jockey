## Installation
```shell
npm install -g streamjockey
```
or
```shell
git clone git@github.com:nogizhopaboroda/streamjockey.git
cd streamjockey
npm install
```

## Usage

***
```shell
streamjockey -h

        -v, --version         output the version number
```
